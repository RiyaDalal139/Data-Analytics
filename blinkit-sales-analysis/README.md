# Blinkit Grocery Sales Analysis

**Tools:** Power BI | SQL | Excel

## Overview
Analysed Blinkit grocery sales data to uncover outlet performance, 
customer ratings, and product visibility trends.

## Key Insights
- Built Power BI dashboard showing total sales, average order value, 
  item visibility scores, and customer ratings
- Segmented data by outlet type, size, and location
- Used SQL to extract and clean raw sales data before analysis
